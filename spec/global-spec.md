# Глобальная Спецификация Проекта Doctor-Amal

## Обзор Проекта
**Название Проекта:** Doctor-Amal  
**Цель:** Локальная платформа для продажи образовательных курсов по поддержке здоровья женщин, питанию и образу жизни  
**Целевая Аудитория:** Женщины в возрасте 12+ лет, русскоязычные пользователи по всему миру, которым интересны темы здоровья, питания и образа жизни  
**Области Фокуса:** Курсы, материалы и предварительная заявка на подбор подходящего курса  

## Методология Разработки
- **Разработка на Основе Спецификаций:** Все функции и требования документируются в спецификациях перед реализацией
- **Локальная Разработка:** 100% оффлайн, без онлайн-компонентов на начальном этапе
- **Будущие Планы:** Онлайн-версия с интеграцией базы данных будет рассмотрена после завершения локальной версии

## Основные Компоненты
### 1. Управление Каталогом Курсов
- Каталог доступных курсов по питанию, поддержке здоровья и образу жизни
- Описания курсов с ценами и характеристиками
- Категоризация курсов по темам

### 2. Пользовательский Интерфейс
- Интерфейс на русском языке
- Дизайн, подходящий по возрасту (12+)
- Интуитивная навигация для просмотра и покупки курсов

### 3. Структура Данных
- Локальное хранение информации о курсах
- Локальное хранение заявок на покупку курса
- Локальное хранение заявок на предварительную консультацию для подбора курса
- Организация курсов по категориям
- Функции поиска и фильтрации курсов

### 4. Предварительная консультация
- Форма заявки для сбора информации о проблеме пользователя
- Используется для подбора подходящего курса перед покупкой
- Не является самостоятельной онлайн-консультацией или заменой очного осмотра

### 5. Telegram-материалы / статьи
- Раздел с бесплатными полезными материалами от доктора
- Дублирование статей и публикаций из Telegram-канала
- Помощь пользователю в знакомстве с подходом доктора до покупки курса

## Технические Требования
- **Платформа:** Десктопное приложение (локальное)
- **Язык:** Русский
- **Технологический Стек:** Будет определён в детальных спецификациях
- **Оффлайн Возможности:** Полная функциональность без интернета

## Цели Проекта
1. Предоставить локальную платформу для продажи образовательных курсов по поддержке здоровья для русскоязычных женщин
2. Обеспечить удобный доступ к курсам по питанию, профилактике и образу жизни
3. Создать каталог курсов для личного и семейного использования
4. Гарантировать конфиденциальность и локальный контроль данных

## Граница Ответственности
- Приложение не ставит диагноз автоматически
- Материалы курсов носят образовательный и информационный характер
- Рекомендации по здоровью должны даваться доктором с учётом состояния пользователя
- При серьёзных, острых или опасных симптомах пользователь должен обратиться за очной медицинской помощью

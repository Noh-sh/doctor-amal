# План 022: Показывать цену в свернутом курсе

## Статус

Завершен.

## Цель

Уточнить `courses-block.md`: в свернутом списке курсов пользователь видит название курса и цену, а после нажатия на название раскрываются описание, цена и кнопка `Купить`.

## Подтверждение specs

Основание:

- Подтверждение владельца проекта в текущей задаче.
- `spec/functional-map.md`
- `spec/feature-specs/courses-block.md`

Подтверждено:

- В блоке `Курсы` сначала видны названия курсов и цена.
- После нажатия на название курса раскрываются описание курса, цена и кнопка `Купить`.

Вне specs:

- Конкретные курсы, цены, валюта и Telegram-ссылка пока не подтверждены.

## Что делаем

1. Обновить описание свернутого состояния курса.
2. Обновить действия пользователя.
3. Обновить основное поведение раскрытия курса.
4. Сохранить запрет на отдельные маршруты, формы, оплату и сбор данных.

## Что не делаем в этом плане

- Не добавляем реальные курсы, цены и ссылки.
- Не меняем код приложения.
- Не выполняем git commit и push.

## Проверка

Команды:

```bash
rg -n "название курса и цену|названия курсов и цены|описание курса, цену и кнопку|Купить|Telegram|раскры" spec/feature-specs/courses-block.md
git diff --check
```

Ручная проверка:

- Убедиться, что свернутое состояние показывает название и цену.
- Убедиться, что раскрытое состояние показывает описание, цену и `Купить`.

## Критерии готовности

- Сценарий отображения цены в свернутом курсе зафиксирован.
- Первая локальная версия не расширена формами, оплатой или сбором данных.
- Результат проверки записан в план.

## Результат проверки

Выполнены проверки:

```bash
rg -n "свернут|название курса|цена курса|названиями и ценами|описание курса|Купить|Telegram|раскры|форм|оплат|данн" spec/feature-specs/courses-block.md
git diff --check
```

Результат:

- В свернутом виде курса показываются название и цена.
- После нажатия на название курса раскрываются описание, цена и кнопка `Купить`.
- Переход по кнопке `Купить` остается переходом в Telegram для связи с менеджером.
- Формы, оплата, отдельные маршруты и сбор данных не добавлены.
- `git diff --check` выполнен без ошибок.

# План 018: Убрать онлайн-консультации из doctor-block

## Статус

Завершен.

## Цель

Уточнить будущие расширения блока `О докторе`: убрать интеграцию с онлайн-консультациями, потому что ожидаемое будущее развитие проекта связано с онлайн-покупкой курса, а не с консультациями.

## Подтверждение specs

Основание:

- Подтверждение владельца проекта в текущей задаче.
- `spec/global-spec.md`
- `spec/functional-map.md`
- `spec/feature-specs/doctor-block.md`

Подтверждено:

- Интеграция с онлайн-консультациями не нужна как будущее расширение блока `О докторе`.
- Максимальное ожидаемое будущее развитие связано с онлайн-покупкой курса: выбор курса, ввод данных, оплата картой, доступ или переход в закрытый Telegram-канал после успешной оплаты.

Вне specs:

- Детали будущей онлайн-покупки не реализуются и не переносятся в первую локальную версию.

## Что делаем

1. Убрать из `spec/feature-specs/doctor-block.md` пункт про интеграцию с онлайн-консультациями.
2. Уточнить, что будущие расширения блока `О докторе` не меняют границы первой локальной версии.

## Что не делаем в этом плане

- Не меняем код приложения.
- Не добавляем формы, оплату или консультации.
- Не выполняем git commit и push.

## Проверка

Команды:

```bash
rg -n "онлайн-консультац|онлайн-покуп|закрытый Telegram|О докторе" spec/feature-specs/doctor-block.md spec/functional-map.md
git diff --check
```

Ручная проверка:

- Убедиться, что `doctor-block.md` не обещает онлайн-консультации.
- Убедиться, что будущая онлайн-покупка остается вне первой локальной версии.

## Критерии готовности

- В `doctor-block.md` нет интеграции с онлайн-консультациями.
- Медицинские и продуктовые границы первой версии сохранены.
- Результат проверки записан в план.

## Результат проверки

Выполнены проверки:

```bash
rg -n "онлайн-консультац|форма записи|онлайн-покуп|закрытый Telegram|О докторе|покупкой курса" spec/feature-specs/doctor-block.md spec/functional-map.md
git diff --check
```

Результат:

- Из будущих расширений блока `О докторе` убраны форма записи или связи и интеграция с онлайн-консультациями.
- Онлайн-консультация осталась только в списке того, что не входит в первую локальную версию.
- Будущая онлайн-версия описана через покупку курса внутри проекта, ввод данных, оплату картой и доступ или переход в закрытый Telegram-канал курса.
- `git diff --check` выполнен без ошибок.

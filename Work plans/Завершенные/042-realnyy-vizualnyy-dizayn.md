# План 042: Реальный визуальный дизайн Taplink-страницы

## Статус

Завершено.

## Цель

Перенести утвержденный тестовый визуальный вариант в реальный дизайн первой Taplink-страницы.

## Подтверждение specs

Основание:

- `spec/functional-map.md`
- `spec/technical-specs/routing-and-ui.md`
- `Design map/test-design-map.md`

Подтверждено:

- первая версия остается одной страницей `/`;
- финальное расположение блоков может уточняться при UI-реализации, если не меняются сценарии и не добавляется новое поведение;
- визуальный вариант был вручную проверен и подтвержден владельцем проекта;
- перенос тестовых решений в настоящий дизайн возможен без изменения продуктового сценария.

Вне specs:

- новый продуктовый сценарий не добавляется.

## Что делаем

1. Обновить визуальные стили страницы под спокойный green nutrition Taplink с restrained liquid glass.
2. Сохранить текущую структуру, данные, кнопки, accordion и неактивные состояния.
3. Добавить иконки, hover-состояния и микроанимации без изменения сценариев.
4. Уточнить layout: крупный avatar, имя `Dr.Amal`, медицинское предупреждение внизу страницы и более компактные блоки.
5. Перевести дизайн из тестового статуса в реальный после подтверждения владельца проекта.
6. Проверить сборку и базовое качество diff.
7. Зафиксировать результат проверки в плане.

## Что не делаем в этом плане

- Не добавляем новые маршруты.
- Не добавляем формы, оплату, корзину, заявки или сбор данных.
- Не добавляем неподтвержденные фото, курсы, цены или URL.
- Не добавляем Supabase, админку или будущие функции.

## Проверка

Команды:

```bash
npm run build
git diff --check
```

Ручная проверка:

- главная страница открывается на `/`;
- блоки `О докторе` и `Курсы` раскрываются на этой же странице;
- внешние кнопки без URL остаются неактивными;
- кнопка `Купить` без Telegram-ссылки остается неактивной;
- на мобильной ширине нет горизонтального скролла.

## Критерии готовности

- Реальный визуальный дизайн первой Taplink-страницы можно посмотреть на странице.
- Поведение первой версии не изменено.
- Проверка выполнена и записана в план.

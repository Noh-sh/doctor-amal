# План 016: Единый текст кнопки "Купить"

## Статус

Завершен.

## Цель

Зафиксировать единый текст кнопки покупки курса в первой версии: `Купить`.

## Подтверждение specs

Основание:

- Подтверждение владельца проекта в текущей задаче.
- `spec/functional-map.md`

Подтверждено:

- В первой версии для покупки курса используется текст кнопки `Купить`.
- Вариант `Приобрести` больше не используется как равноправный вариант.

Вне specs:

- Остальные детали блока курсов будут уточняться в feature specs.

## Что делаем

1. Заменить формулировки `Купить` или `Приобрести` на `Купить` в `spec/functional-map.md`.
2. Обновить актуальные записи Roadmap.

## Что не делаем в этом плане

- Не меняем код приложения.
- Не уточняем структуру курса, цены, ссылки и способ раскрытия курса.
- Не выполняем git commit и push.

## Проверка

Команды:

```bash
rg -n "Приобрести|Купить" spec/functional-map.md Roadmap/chronology.md Roadmap/project-roadmap.md
git diff --check
```

Ручная проверка:

- Убедиться, что в текущих specs используется только `Купить`.
- Убедиться, что остальные открытые вопросы оставлены для feature specs.

## Критерии готовности

- В `functional-map.md` не осталось варианта `Приобрести`.
- В Roadmap нет актуальной записи, где `Приобрести` указано как вариант текущей первой версии.
- Результат проверки записан в план.

## Результат проверки

Выполнены проверки:

```bash
rg -n "Приобрести|Купить" spec/functional-map.md Roadmap/chronology.md Roadmap/project-roadmap.md "Work plans/Завершенные/016-edinyy-tekst-knopki-kupit.md"
git diff --check
```

Результат:

- В `spec/functional-map.md` используется только кнопка `Купить`.
- В актуальных записях Roadmap кнопка покупки указана как `Купить`.
- Вариант `Приобрести` остался только в тексте этого рабочего плана как описание замененной формулировки.
- `git diff --check` выполнен без ошибок.

## Измененные файлы

- `spec/functional-map.md`
- `Roadmap/chronology.md`
- `Roadmap/project-roadmap.md`
- `Work plans/Завершенные/016-edinyy-tekst-knopki-kupit.md`

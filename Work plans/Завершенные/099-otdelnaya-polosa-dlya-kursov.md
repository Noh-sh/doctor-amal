# План 099: Отдельная продуктовая полоса для курсов

## Статус

Завершен.

## Цель

Сделать для второй вкладки `Курсы` отдельную декоративную food-полосу с другим набором овощей и фруктов, включая целое и разрезанное авокадо.

## Подтверждение specs

Основание:

- `spec/technical-specs/routing-and-ui.md`;
- `Design map/test-design-map.md`;
- `Design map/design-rules.md`.

Подтверждено:

- допустимы небольшие растительные и нутрициологические акценты;
- UI-детали можно менять без изменения данных, маршрутов и пользовательского сценария;
- food-изображение не должно мешать чтению или создавать медицинское обещание.

Вне specs:

- новых функций, данных, текстов и сценариев нет.

## Что делаем

1. Создать отдельный оригинальный asset для вкладки `Курсы`.
2. Использовать другой набор продуктов с целым и разрезанным авокадо.
3. Назначить первую полосу блоку `О докторе`, вторую — блоку `Курсы`.
4. Выполнить проверки.

## Что не делаем

- Не меняем тексты, данные, маршруты и accordion-поведение.
- Не добавляем таблетки, добавки и медицинские предметы.
- Не выполняем commit или push.

## Проверка

```bash
npm run quality
git diff --check
```

## Результат проверки

- Через встроенный `imagegen` создан отдельный asset для блока `Курсы`.
- В композиции есть целый и разрезанный авокадо, лайм, груша, инжир, виноград, горошек и зелень.
- Первая полоса сохранена для блока `О докторе`, новая назначена только блоку `Курсы`.
- `npm run quality` выполнен успешно: прошли `lint`, `typecheck` и production-сборка Next.js.
- `git diff --check` выполнен без ошибок.

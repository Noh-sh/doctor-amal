# План 023: Убрать цену из раскрытого курса

## Статус

Завершен.

## Цель

Уточнить `courses-block.md`: цена курса видна в свернутом состоянии рядом с названием, поэтому в раскрытом состоянии остаются только описание курса и кнопка `Купить`.

## Подтверждение specs

Основание:

- Подтверждение владельца проекта в текущей задаче.
- `spec/feature-specs/courses-block.md`

Подтверждено:

- Цена курса показывается в свернутом виде рядом с названием курса.
- После нажатия на название курса раскрываются описание курса и кнопка `Купить`.
- Дублировать цену в раскрытом состоянии не нужно.

Вне specs:

- Конкретные курсы, цены, валюта и Telegram-ссылка пока не подтверждены.

## Что делаем

1. Убрать цену из описания раскрытого состояния курса.
2. Обновить действия пользователя и основное поведение.
3. Сохранить запрет на формы, оплату и сбор данных.

## Что не делаем в этом плане

- Не добавляем реальные курсы, цены и ссылки.
- Не меняем код приложения.
- Не выполняем git commit и push.

## Проверка

Команды:

```bash
rg -n "свернут|цена курса|описание курса|кнопка `Купить`|раскры|описание курса, цену|описание, цена" spec/feature-specs/courses-block.md
git diff --check
```

Ручная проверка:

- Убедиться, что цена есть в свернутом состоянии.
- Убедиться, что раскрытое состояние не дублирует цену.

## Критерии готовности

- Цена курса остается в свернутом состоянии.
- Раскрытое состояние содержит описание курса и кнопку `Купить`.
- Результат проверки записан в план.

Результат:

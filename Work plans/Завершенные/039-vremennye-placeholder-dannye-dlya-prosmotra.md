# План 039: Временные placeholder-данные для просмотра UI

## Статус

Завершено.

## Цель

Добавить временные визуальные placeholder-данные и проверить более удобное раскрытие блоков `О докторе` и `Курсы` прямо под кнопками.

## Границы

- Placeholder-данные не являются реальным контентом.
- Нельзя придумывать реальные курсы, цены, биографию или медицинские утверждения.
- После визуальной проверки placeholder-данные нужно убрать или заменить подтвержденными данными.
- Раскрытие `О докторе` и `Курсы` должно оставаться одностраничным поведением без новых маршрутов.

## Проверка

- Добавлены временные placeholder-данные для блока `О докторе` и двух курсов.
- Блоки `О докторе` и `Курсы` раскрываются inline под соответствующей кнопкой.
- Повторное нажатие на кнопку закрывает раскрытый блок.
- `npm run build` выполнен успешно.
- `git diff --check` выполнен без ошибок.

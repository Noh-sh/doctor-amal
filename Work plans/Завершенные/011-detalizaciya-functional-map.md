# План 011: Детализация functional map для Taplink-страницы

## Статус

Завершен.

## Цель

Уточнить `spec/functional-map.md` под подтвержденную первую версию: одна локальная страница с фото и именем доктора, набором кнопок и переходами на внешние платформы.

## Подтверждение specs

Основание:

- Подтверждение владельца проекта в текущей задаче.
- `spec/global-spec.md`
- `spec/functional-map.md`
- `spec/technical-specs/change-management.md`

Подтверждено:

- Проект состоит из одной страницы.
- В начале страницы должны быть фото и имя доктора.
- В первой версии нужны кнопки: `О докторе`, `Telegram`, `WhatsApp`, `YouTube`, `Instagram`, `Курсы`.
- Внутри проекта не будет медицинских текстов.
- Проект не собирает данные пользователя.
- Первая версия является локальной.
- В будущем проект должен перейти в онлайн.

Вне specs:

- Конкретные URL для кнопок пока не подтверждены.
- Дизайн, верстка и код в этом плане не меняются.

## Что делаем

1. Обновить `spec/functional-map.md` под подтвержденную структуру одной страницы.
2. Зафиксировать список кнопок первой версии.
3. Зафиксировать отсутствие медицинских текстов и сбора данных.
4. Зафиксировать будущий переход проекта в онлайн без реализации в первой версии.

## Что не делаем в этом плане

- Не меняем код приложения.
- Не добавляем реальные ссылки без подтверждения.
- Не создаем внутренние маршруты.
- Не выполняем git commit и push.

## Проверка

Команды:

```bash
rg -n "О докторе|Telegram|WhatsApp|YouTube|Instagram|Курсы|медицинских текстов|данные пользователя|одной страницы|онлайн" spec/functional-map.md
git diff --check
```

Ручная проверка:

- Убедиться, что `functional-map.md` описывает одну страницу.
- Убедиться, что список кнопок первой версии зафиксирован.
- Убедиться, что `О докторе` не превращен в отдельную страницу.
- Убедиться, что сбор данных и медицинские тексты исключены из первой версии.

## Критерии готовности

- Functional map соответствует подтвержденным требованиям.
- В документе нет внутреннего каталога курсов как функции первой версии.
- В документе нет форм, заявок или сбора персональных данных.
- Результат проверки записан в план.

Выполнены проверки:

# План 098: Декоративные фруктовые полосы

## Статус

Завершен.

## Цель

Добавить в тестовый дизайн публичной страницы небольшие декоративные фото-полосы с фруктами и овощами по предоставленному референсу.

## Подтверждение specs

Основание:

- `spec/functional-map.md`;
- `spec/technical-specs/routing-and-ui.md`;
- `Design map/test-design-map.md`;
- `Design map/design-rules.md`.

Подтверждено:

- допустимы растительные и нутрициологические визуальные акценты;
- декоративные изображения не должны мешать тексту или создавать ощущение магазина;
- тестовый дизайн может менять композицию без изменения сценариев и данных.

Вне specs:

- новых функций, данных, текстов и сценариев нет.

## Что делаем

1. Создать оригинальный декоративный food-asset без текста, таблеток и медицинских предметов.
2. Добавить узкие полосы к внутренним блокам `О докторе` и `Курсы`.
3. Проверить адаптивность и production-сборку.

## Что не делаем

- Не используем изображение как фон под текстом.
- Не меняем тексты, данные, маршруты и поведение accordion.
- Не выполняем commit или push.

## Проверка

```bash
npm run quality
git diff --check
```

## Критерии готовности

- Полосы выглядят как спокойный декоративный акцент.
- Текст и кнопки остаются читаемыми.
- Проверки проходят.

## Результат проверки

- Через встроенный `imagegen` создан оригинальный декоративный asset без текста, людей, таблеток, баночек и медицинских предметов.
- Asset сохранен в проекте как `public/images/nutritionStrip.png`.
- `npm run quality` выполнен успешно: прошли `lint`, `typecheck` и production-сборка Next.js.
- `git diff --check` выполнен без ошибок.
